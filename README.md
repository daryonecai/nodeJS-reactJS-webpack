# sailsProject2

一、sailsJs框架目录结构:
# api
# controllers(控制器相关，业务逻辑)
# models(模块开发)
# policies(用于路由过滤)
# responses(定制所需的响应:如404 res.notFound())
# services(定制一些常用的工具类–全局的)
# assets(静态资源文件)
# images
# js
# styles
# templates
# favicon.ico
# config(整个项目的配置文件系统)
# env(配置不同环境的变量)
# *.js
# tasks
# views

二、项目描述：
# 此DEMO,nodejs + sailsJs + reactJs + webpack，并支持ES6,webpack打包工具。

三、安装包

# 1、cd sailsProject2
# 2、在项目目录下npm install

# 运行：node app.js
# 访问：http://localhost:1337
